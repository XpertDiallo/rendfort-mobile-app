# Contenu Pedagogique

Ce dossier contient les sources et contenus structures de RENDFORT.

## Dossiers

- `programme_2c/official` : referentiel officiel extrait et structure.
- `programme_2c/lessons` : lecons originales RENDFORT.
- `programme_2c/quizzes` : quiz lies aux lecons.
- `sources_autorisees/raw` : references brutes locales, non publiees.
- `sources_autorisees/ocr` : sorties OCR.
- `sources_autorisees/cleaned` : textes nettoyes pour analyse interne.
- `sources_autorisees/mapping` : correspondance entre supports autorises et programme officiel.
- `generated` : donnees de seed et fixtures.

## Principe

Le programme officiel pilote la structure. Les supports autorises aident l'analyse et la production pedagogique.

Dans l'application, ne jamais afficher de termes techniques internes ni de noms de sources internes.
