# Sources et Autorisations

## Programme officiel

Le programme educatif `06.Prog Educt maths 2C CND 0923.pdf` est la source maitresse pour structurer l'application.

## Cours adopte pour le MVP

Le fichier `cours maths apc 2nde C ecole-online ci.pdf` est le support principal retenu pour construire les lecons, les sections de cours et les quiz de la V1.

## Supports de reference autorises internes

Les supports secondaires fournis au projet sont des supports scannes dont l'utilisation est autorisee.

Ces noms de sources restent des references internes. Ils ne doivent jamais apparaitre dans l'application mobile, dans les contenus visibles par les eleves, dans les emails, dans les notifications ni dans les messages d'erreur.

## Regle de production de contenu

Le contenu RENDFORT doit etre aligne sur le programme officiel, conforme aux autorisations obtenues et valide pedagogiquement.

## Regle de vocabulaire application

Les termes techniques internes et les noms de sources internes sont interdits dans l'application.

Termes autorises cote utilisateur :

- `tuteur RENDFORT`
- `prof RENDFORT`
- `assistant RENDFORT`
- `support de cours`
- `manuel de reference`
- `guide pedagogique`
- `assistance interactive`
