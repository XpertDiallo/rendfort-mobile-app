# API RENDFORT

Backend cible du MVP V1.

Stack recommandee :

- FastAPI.
- PostgreSQL.
- pgvector.
- SQLAlchemy ou SQLModel.
- Alembic.

Modules prevus :

- `auth` : inscription, verification email, login, refresh token.
- `content` : syllabus, lecons, sections.
- `progress` : progression eleve.
- `quiz` : quiz, tentatives, scores.
- `tutor` : tuteur contextualise.

Ce dossier ne contient pas encore l'application FastAPI initialisee.
